Docker & Kubernetes, fondamentaux 10 Les commandes Dockerfile (3/3) • WORKDIR : elle définit le répertoire de travail à utiliser pour le lancement des commandes CMD et/ou ENTRYPOINT et ça sera aussi le dossier courant lors du démarrage du conteneur. • EXPOSE : elle expose un port. • VOLUMES : elle crée un point de montage qui permettra de persister les données. • USER : elle désigne quel est l'utilisateur qui lancera les prochaines instructions RUN, CMD ou ENTRYPOINT (par défaut c'est l'utilisateur root). • et bien d’autres…  https://docs.docker.com/reference/dockerfile /
